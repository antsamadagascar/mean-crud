# Application MEAN - CRUD Articles

Application simple de gestion d'articles avec le stack MEAN (MongoDB, Express, Angular, Node.js).

---

##  Fonctionnalités

- Créer, lire, modifier et supprimer des articles
- Interface Angular réactive
- API RESTful avec Express
- Base de données MongoDB

---

## Prérequis

- Node.js v20+
- Angular CLI v17+
- MongoDB v8+

---

## Installation rapide

### 1. Cloner le projet

```bash
git clone https://github.com/antsamadagascar/mean-crud.git
cd mean-crud
```

### 2. Backend

```bash
cd backend
npm install
npm run dev
```

### 3. Frontend

```bash
cd frontend
npm install
ng serve --open
```

### 4. MongoDB

```bash
# Lancer MongoDB
mongod --dbpath /var/lib/mongodb
```

---

##  URLs de l'application

- **Frontend** : http://localhost:4200
- **Backend** : http://localhost:5000
